Παράδειγμα Λίστες Ορισμών

Ορισμός Κοινωνία της Πληροφορίας
Η κοινωνία, στην οποία κυριαρχεί η παραγωγή, διακίνηση, διαχείριση, αξιοποίηση και ενσωμάτωση της πληροφορίας.

Ορισμός Βιβλιοθηκομίας
Ένα σύνολο κανόνων βάση των οποίων συγκεντρώνεται η γνώση και οργανώνονται και διοικούνται οι βιβλιοθήκες με σκοπό να διασώσουν τη γνώση για τους χρήστες της.

Ορισμός HTML
H HTML είναι ο τυπικός κώδικας σήμανσης που χρησιμοποιείται για τη δημιουργία και τη δομή ιστοσελίδων στο Διαδίκτυο.

Ορισμός SQL
Η SQL είναι μία γλώσσα υπολογιστών στις βάσεις δεδομένων, που σχεδιάστηκε για τη διαχείριση δεδομένων.

Ορισμός AACR
Οι Αγγλοαμερικανικοί κανόνες καταλογογράφησης ACCR2 αποτελούν ένα εθνικό κώδικα καταλογογράφησης, οι οποίοι δημοσιεύθηκαν για πρώτη φορά το 1967.
